## 1. 项目概述

### 1.1 项目背景

开发一套支持多角色（学生/教师/管理员）的在线教学平台，支持PDF文档分发、在线问答、自动化评分及成绩管理。

### 1.2 技术栈

| 层级 | 技术 | 用途 |
|------|------|------|
| 前端 | Vue 3 + Element Plus / Ant Design Vue | 三端Web界面 |
| 后端 | Django + Django REST Framework | API服务 |
| 数据库 | MySQL 8.0 | 数据持久化 |
| 实时通信 | Django Channels + WebSocket | 实时推送成绩更新 |
| 文件存储 | 本地/MinIO/OSS | PDF文档存储 |
| 评分脚本 | Python/Bash (Linux客户端) | 自动化评测 |

## 3. 功能需求

### 3.1 登录模块

#### 3.1.1 登录方式

- **FR-L-001**: 学生使用**学号**和密码登录
- **FR-L-002**: 教师使用**教师号**和密码登录
- **FR-L-003**: 管理员使用**教师号**和密码登录（管理员本质是教师角色提升）
- **FR-L-004**: 登录时系统根据输入的账号自动识别角色（学号→学生，教师号→教师/管理员）
- **FR-L-005**: 登录成功后返回JWT Token，有效期2小时，支持刷新

#### 3.1.2 找回密码

- **FR-L-006**: 登录页面增加"忘记密码"入口
- **FR-L-007**: 找回密码流程：
  1. 输入绑定的手机号
  2. 获取短信验证码
  3. 验证通过后，设置新密码
  4. 密码重置成功，自动跳转登录页
- **FR-L-008**: 验证码有效期5分钟，错误次数限制3次
- **FR-L-009**: 新密码不能与旧密码相同

### 3.2 学生端功能

#### 3.2.1 节PDF查看

- **FR-S-001**: 学生登录后进入当前激活的节，自动加载教师发布的PDF
- **FR-S-002**: 支持PDF在线预览（基于PDF.js），无需下载
- **FR-S-003**: 支持PDF缩放、翻页、目录导航
- **FR-S-004**: 历史节的PDF可回顾，但默认打开当前激活的节

#### 3.2.2 问题回答

- **FR-S-005**: 查看当前节下教师发布的题目列表
- **FR-S-006**: 支持多种题型：单选、多选、填空、简答、编程题
- **FR-S-007**: 编程题支持代码编辑器（Monaco Editor），支持语法高亮
- **FR-S-008**: 提交答案后，对于客观题即时显示正误；主观题/编程题显示"评分中"状态
- **FR-S-009**: **反复提交**: 在教师设置的提交截止日期前，学生可反复修改并重新提交答案
- **FR-S-010**: 超过截止日期后，答案锁定不可修改（根据教师配置）

#### 3.2.3 成绩查看

- **FR-S-011**: 实时查看当前节各题目得分及总分
- **FR-S-012**: 成绩更新时（评分脚本返回结果），页面自动刷新成绩显示
- **FR-S-013**: 查看历史节成绩

## 5. 数据库设计

### 5.1 用户与权限

```sql
-- 用户基表
CREATE TABLE users (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    password VARCHAR(255) NOT NULL,              -- bcrypt加密
    name VARCHAR(50) NOT NULL,
    phone VARCHAR(20),                           -- 手机号
    phone_verified BOOLEAN DEFAULT FALSE,
    role ENUM('admin','teacher','student') NOT NULL,
    is_active BOOLEAN DEFAULT TRUE,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);

-- 学生表
CREATE TABLE students (
    user_id BIGINT PRIMARY KEY,
    student_no VARCHAR(20) UNIQUE NOT NULL,      -- 学号，学生登录账号
    class_name VARCHAR(50) NOT NULL,             -- 班级
    group_id BIGINT,
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE
);

-- 教师表
CREATE TABLE teachers (
    user_id BIGINT PRIMARY KEY,
    teacher_no VARCHAR(20) UNIQUE NOT NULL,      -- 教师号，教师/管理员登录账号
    department VARCHAR(50),                      -- 所属部门
    is_admin BOOLEAN DEFAULT FALSE,              -- 是否为管理员
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE
);
```

### 5.2 课程结构

```sql
-- 课程表
CREATE TABLE courses (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(100) NOT NULL,
    teacher_id BIGINT NOT NULL,
    description TEXT,
    status ENUM('draft','active','archived') DEFAULT 'draft',
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (teacher_id) REFERENCES users(id)
);

-- 章表
CREATE TABLE chapters (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    course_id BIGINT NOT NULL,
    chapter_no INT NOT NULL,                     -- 章序号
    title VARCHAR(100) NOT NULL,
    sequence INT DEFAULT 0,
    is_visible BOOLEAN DEFAULT TRUE,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (course_id) REFERENCES courses(id) ON DELETE CASCADE
);

-- 节表
CREATE TABLE sections (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    chapter_id BIGINT NOT NULL,
    section_no INT NOT NULL,                     -- 节序号
    title VARCHAR(100) NOT NULL,
    sequence INT DEFAULT 0,
    is_active BOOLEAN DEFAULT FALSE,             -- 是否为当前激活的节
    is_visible BOOLEAN DEFAULT TRUE,
    start_time TIMESTAMP NULL,
    end_time TIMESTAMP NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (chapter_id) REFERENCES chapters(id) ON DELETE CASCADE
);
```

### 5.3 文档与题目

```sql
-- 文档表
CREATE TABLE documents (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    section_id BIGINT NOT NULL,
    title VARCHAR(100) NOT NULL,
    file_path VARCHAR(255) NOT NULL,
    file_size BIGINT,
    uploaded_by BIGINT NOT NULL,
    version INT DEFAULT 1,
    is_visible BOOLEAN DEFAULT TRUE,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (section_id) REFERENCES sections(id) ON DELETE CASCADE,
    FOREIGN KEY (uploaded_by) REFERENCES users(id)
);

-- 题目表
CREATE TABLE questions (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    section_id BIGINT NOT NULL,
    type ENUM('single','multiple','fill','essay','programming','attachment') NOT NULL,
    title TEXT NOT NULL,
    content TEXT,
    options JSON,                                -- 选择题选项
    correct_answer TEXT,                         -- 客观题答案
    test_cases JSON,                             -- 编程题测试用例
    max_score DECIMAL(5,2) DEFAULT 100.00,
    time_limit INT,                              -- 编程题时间限制(ms)
    memory_limit INT,                            -- 编程题内存限制(MB)
    allowed_languages JSON,                      -- ["python","cpp"]
    deadline TIMESTAMP NULL,
    allow_resubmit BOOLEAN DEFAULT TRUE,         -- 是否允许重新提交
    is_published BOOLEAN DEFAULT FALSE,
    order_num INT DEFAULT 0,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (section_id) REFERENCES sections(id) ON DELETE CASCADE
);
```

### 5.4 成绩与评分

```sql
-- 成绩表
CREATE TABLE scores (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    student_id BIGINT NOT NULL,
    student_no VARCHAR(20) NOT NULL,             -- 学号（冗余存储）
    class_name VARCHAR(50),                      -- 班级（冗余存储）
    student_name VARCHAR(50),                    -- 姓名（冗余存储）
    chapter_no INT NOT NULL,
    chapter_name VARCHAR(100),
    section_no INT NOT NULL,
    section_name VARCHAR(100),
    section_id BIGINT NOT NULL,
    score DECIMAL(6,2) NOT NULL,

    source ENUM('auto_script','manual','import') DEFAULT 'auto_script',
    evaluator VARCHAR(100),                      -- 评分机标识或教师姓名
    details TEXT,                                -- 评分详情

    original_score DECIMAL(6,2),                 -- 原始分数
    modified_by BIGINT,
    modified_at TIMESTAMP NULL,
    modify_reason VARCHAR(255),

    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,

    UNIQUE KEY uk_student_section (student_id, section_id),
    FOREIGN KEY (student_id) REFERENCES users(id),
    FOREIGN KEY (section_id) REFERENCES sections(id),
    FOREIGN KEY (modified_by) REFERENCES users(id)
);

-- 成绩修改历史表
CREATE TABLE score_history (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    score_id BIGINT NOT NULL,
    old_score DECIMAL(6,2) NOT NULL,
    new_score DECIMAL(6,2) NOT NULL,
    modified_by BIGINT NOT NULL,
    reason VARCHAR(255),
    modified_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (score_id) REFERENCES scores(id) ON DELETE CASCADE,
    FOREIGN KEY (modified_by) REFERENCES users(id)
);
```

### 5.5 答案提交

```sql
-- 答案提交表
CREATE TABLE submissions (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    student_id BIGINT NOT NULL,
    question_id BIGINT NOT NULL,
    section_id BIGINT NOT NULL,
    answer TEXT,                                 -- 答案内容（代码/文本）
    language VARCHAR(20),                        -- 编程题语言
    submitted_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    ip_address VARCHAR(45),
    FOREIGN KEY (student_id) REFERENCES users(id),
    FOREIGN KEY (question_id) REFERENCES questions(id),
    FOREIGN KEY (section_id) REFERENCES sections(id)
);
```

## 8. 开发阶段建议

| 阶段 | 周期 | 交付物 |
|------|------|--------|
| **Phase 1** | 2周 | 基础框架：用户认证（学号/教师号登录）、角色权限、数据库搭建、手机号绑定/找回密码 |
| **Phase 2** | 2周 | 教师端：学生管理（搜索+班级筛选）、课程管理（章-节结构）、PDF发布 |
| **Phase 3** | 2周 | 学生端：PDF查看、题目作答（反复提交）、WebSocket成绩推送 |
| **Phase 4** | 2周 | 评分系统：评分API（覆盖逻辑）、Linux脚本对接、成绩入库推送 |
| **Phase 5** | 2周 | 成绩管理：Excel导入导出、排序功能、修改审计 |
| **Phase 6** | 1周 | 管理员端：教师管理、权限设置 |
| **Phase 7** | 1周 | 测试优化、部署文档 |
